# Historical Documentation Archive

This directory preserves dated design specifications and implementation plans.
They are kept for decision history and may include Railway, NFT Gym, or other
retired assumptions.

They are **not** the operational source of truth. Use the maintained documents
listed in [the documentation map](../README.md) for the current general
OneBrain and Hetzner-only architecture.

- `specs/` contains design records.
- `plans/` contains implementation records.

Archive files retain their original names and content. Any path updates made as
part of archival are mechanical only.
